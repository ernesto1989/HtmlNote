NOTAS SOBRE HTML

Html: Hypertext Markup Language.

Hypertext: Es la parte de HTML que permite crear hypertexto o hyperlinks. Son enlaces a otros documentos HTML.

Markup: Los markups son un concepto que viene de la imprenta, cuando se usaban "markups" para brindar information sobre como dar formato al texto escrito.

Los documentos HTML permiten generar la estructura general de documentos o aplicaciones. Se convina con Javascript y CSS para generar aplicaciones web con funcionalidad definida y un diseño especializado.
