La etiqueta html

Las etiquetas html indica al navegador que todo lo que esté entre ellas debe ser interpretado cómo html. Normalmente dentro de html hay 2 cosas:
  • <head></head>: Provee información (metadata) para indicar al navegador información sobre la página.
  • <body></body>:Tiene el contenido de la página html 
    <!DOCTYPE html>
    <html>
      <head>
        <meta charset="utf-8">
        <title> </title>
      </head>
      <body>
        contenido html
      </body>
    </html>

Etiquetas meta

Dentro del header, las etiquetas meta brindan información a dicional al navegador

Títulos con los tag h

Los títulos se escriben con los tags <h1> a <h6> 
    <h1> Título con h1 </h1>
    <h2> Título con h2 </h2>
    <h3> Título con h3 </h3>
    <h4> Título con h4 </h4>
    <h5> Título con h5 </h5>
    <h6> Título con h6 </h6>

Saltos de linea

Los saltos de linea se pueden hacer en html con el tag <br> 
    Sálto de linea despues de este texto
    <br>

Comentarios

Los comentarios se escriben de la siguiente forma 
    <!-- TODO ESTO ES COMENTADO -->

Las reglas horizontales (lineas de división)

Para imprimir una linea y separar dos secciones, está la etiqueta <hr> 
    Se crea una linea entre esto
    <hr>
    y Esto

Listas

Existen 2 tipos de listas (ordenadas y no ordenadas). Las no ordenadas están compuestas por bulets. Las ordenadas pueden ser numeradas de diferente forma.
Para crear una lista no ordenada basta con las etiquetas: 
    <ul>
      <li>Elemento</li>
      <li>Elemento</li>
      <li>Elemento</li>
    </ul>
Para crear una lista ordenada basta con las etiquetas: 
    <ol>
      <li>Elemento</li>
      <li>Elemento</li>
      <li>Elemento</li>
    </ol>
Se puede utilizar el atributo type en la etiquta ol para cambiar el timpo de numeración.

Anchor tag

Permite crear links (hypertext) a otros documentos html. 
    <a>ESTE ES UN LINK</a>

Tablas

Las tablas tienen 2 propósitos:
  1. Mostrar atributos de contenido (tablas o grid normales)
  2. Dar formato a código html en la pantalla (layout)
Para mostrar una tabla se requiere de 6 etiquetas:
  1. Etiqueta table
  2. Etiqueta thead para definir el titulo
  3. Etiqueta tbody para definir el cuerpo de la tabla
  4. Etiqueta tr para indicar filas de contenido (en thead o tbody)
  5. etiquetas td para indicar el contenido de una celda
  6. En el thead puede sustituirse el td por th para formato de titulo
Para mostrar el margen entre elementos, a la etiqueta table se le da un border 
    <table border="2">
      <thead>
        <tr>
          <th>TITULO 1</th>
          <th>TITULO 2</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Dato 1</td>
          <td>Dato 2</td>
        </tr>
        <tr>
          <td>Dato 3</td>
          <td>Dato 4</td>
        </tr>
      </tbody>
    </table>

          TITULO 1
          TITULO 2
        

      
      
        

          Dato 1
          Dato 2
        

        

          Dato 3
          Dato 4
