Doctype

El doctype permite a los navegadores saber con qué versión de html está escrito el documento.


El doctype de html5 es: 
      <!DOCTYPE html>

El doctype de html4 es: 
      <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
